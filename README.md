# 洞洞板布局工具 (Perfboard Layout Tool)

一个离线可用的洞洞板布局设计工具，单 HTML 文件，浏览器直接打开即用。用于设计洞洞板（perfboard）的元器件布局和走线。

本项目基于 Claude Code + DeepSeek V4 Pro/Flash 开发，总计花费 ¥13.07，173M tokens。本项目的目的主要是考虑到在嵌入式任务前期完成各模块（PCB 板设计及焊接）后，需要对所有模块及单片机（还包括各种接线的排针/排座）进行连接而需要对洞洞板进行设计与焊接，故而涉及到的元器件并不多，因而设计该轻量的洞洞板布局工具，用以进行电子版设计。

## 使用方式

直接双击 `perfboard-v1.1.html` 在浏览器中打开即可。`旧版本/` 文件夹包含全部历史版本（v0.1 ~ v0.20）。

**开发方式**：本项目采用多级文件架构（`src/` 目录），通过 `python build.py vX.X` 合并为单文件分发。开发时编辑 `src/` 下各文件，构建后输出到 `dist/`。

## 功能概览

### 走线
- **焊锡走线**：曼哈顿路径（上下左右），焊锡模式(5)双击开始/单击固定段/双击结束；自动合并相接走线
- **飞线**：飞线模式(4)点击两格点创建虚线连接

### 元器件
- **贴片器件**：贴片模式(2)点击相邻两格点放置，位于格点之间，可标注名称
- **排针/排座**：排针模式(3)拖动矩形放置，每格点对应一个引脚，可逐个标注引脚标签
- **器件编组**：框选后按 G 编组（整体拖动/旋转/翻转），按 U 拆组

### 编辑
- **框选**：选择模式下拖动矩形框选器件
- **焊锡段选中**：点击焊锡段选中（橙色高亮），可拖动移动、Delete 删段（前后分裂）
- **橡皮擦**：橡皮擦模式(6)长按拖拽擦除器件/焊锡段
- **焊锡线属性**：选中焊锡线/段后右侧面板修改颜色和名称
- 器件移动时已连接的焊锡/飞线不会跟随移动

## 文件管理

| 按钮 | 行为 |
|------|------|
| **+ 新建** | 清空画布，自动命名 untitled/untitled2 加入工作区 |
| **📂 打开** | 从电脑选 `.json` 加载到工作区 |
| **💾 保存** / `Ctrl+S` | 更新工作区（不弹窗，不改名，不下载文件） |
| **📥 另存为** | 弹窗命名 + 下载 `.json` 文件 |
| **侧边栏工作区** | 点击文件切换（自动保存当前）、× 删除 |
| **导入文件** | 侧边栏"导入文件"可批量加入工作区 |
| 关闭浏览器时自动存 localStorage，下次打开恢复

## 焊锡模式使用说明

焊锡模式（按 `5` 进入）采用 Multisim 风格的走线规则：

| 操作 | 行为 |
|------|------|
| **双击格点** | 开始走线，预览线跟随鼠标移动 |
| **单击格点** | 固定当前段到 model，自动从终点继续走线 |
| **双击格点** | 提交最后一段并结束走线 |
| **Enter** | 结束当前走线（不提交最后一段） |
| **Escape** | 取消当前走线和所有操作 |

选择模式下长按格点拖动可快速画单段焊锡。
