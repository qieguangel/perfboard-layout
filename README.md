# 洞洞板布局工具 (Perfboard Layout Tool)

一个离线可用的洞洞板布局设计工具，单 HTML 文件，浏览器直接打开即用。

## 功能

- **无限点阵格**：可缩放/平移的网格画布
- **焊锡走线**：曼哈顿路径（上下左右），长按格点拖动走线
- **飞线**：连接任意两格点的虚线
- **贴片器件**：位于两格点之间，可标注名称
- **排针/排座**：矩形区域，每个格点为一个引脚，可逐个标注引脚标签

## 快捷键

| 按键 | 功能 |
|------|------|
| `1` `2` `3` `4` `5` | 模式切换：选择/贴片/排针/飞线/焊锡 |
| 滚轮 | 缩放画布 |
| 右键拖动 | 平移画布 |
| 左键短按 | 选中器件/焊锡段/飞线 |
| 左键长按器件 | 拖动移动器件 |
| 左键长按格点 | 拖动走焊锡 |
| 双击器件 | 编辑名称 |
| 双击排针引脚 | 标注引脚标签 |
| `Ctrl+Z` / `Ctrl+Y` | 撤销 / 重做 |
| `Ctrl+C` / `Ctrl+V` / `Ctrl+X` | 复制 / 粘贴 / 剪切 |
| `Ctrl+R` | 旋转器件 90° |
| `Ctrl+S` | 另存为 JSON 文件 |
| `H` / `V` | 水平 / 垂直翻转 |
| `Delete` | 删除选中对象 |
| `Escape` | 取消操作 |
| `?` | 快捷键帮助面板 |
| `🌓` | 切换深色/浅色主题 |

## 文件管理

- **+ 新建** — 清空画布创建新布局
- **📂 打开** — 从电脑选择 `.json` 文件加载布局
- **💾 另存为** — 输入文件名，下载 `.json` 到下载文件夹
- **📥 导出** — 直接下载 JSON 文件
- 关闭浏览器时自动保存到 localStorage，下次打开自动恢复

## 使用方式

直接双击 `perfboard.html` 或 `perfboard-v0.5.html` 在浏览器中打开即可。
